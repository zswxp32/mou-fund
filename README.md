# 大牛基金助手（Mou Fund）
大牛基金助手，实时查看您关注的基金，助您快速获取实时数据

## 目的
- 为了上班的时候不用拿起手机查看基金涨跌情况
- 为了你的一个 Star，请不要吝啬，撸代码也需要挤时间的

## 功能
- 基金持仓涨跌角标，实时更新（数据来源：天天基金）
- 大盘数据，实时更新
- 基金添加、配置
- 自动计算涨跌幅度和持仓收益
- 面板展示基金实时涨跌走势图`（ 独有 ）`
- 基金重仓股票实时涨跌走势图
- 标准/简洁模式切换
- 可手动刷新

## 使用方式
- chrome extension 安装 `大牛基金助手`
- 通过基金代码或名称搜索并添加基金
- 在编辑中输入持有份额和持仓成本，enjoy
